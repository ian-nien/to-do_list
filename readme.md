

### Todo List 
A todo list for adding, deleting, updating, prioritzing items.


* `git clone `
* `cd to-do_list`

Please open two terminal to run server and client simultaneously.

## Run Server
1. `cd server`
2. `npm install`
3. `npm run build`
4. `npm start`

## Run Client
1. `cd client`
2. `npm install`
3. `npm start`
